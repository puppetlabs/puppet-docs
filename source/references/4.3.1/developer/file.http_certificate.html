Certificate

The certificate endpoint returns the certificate for the specified name, which might be either a standard certname or ca.

Find

Get a certificate.

GET /puppet-ca/v1/certificate/:nodename?environment=:environment

Supported HTTP Methods

GET

Supported Response Formats

s (denotes a string of text)

The returned certificate is always in the .pem format.

Parameters

None

Notes

The environment field is ignored.

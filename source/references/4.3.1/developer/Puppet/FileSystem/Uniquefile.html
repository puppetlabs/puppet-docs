Overview

A class that provides Tempfile-like capabilities, but does not attempt to manage the deletion of the file for you. API is identical to the normal Tempfile class.

def self.open_tmp(identifier)
  f = new(identifier)
  yield f
ensure
  if f
    f.close!
  end
end
